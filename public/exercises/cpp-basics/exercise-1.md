# Exercise 1: C++ Basics

## Objective
The goal of this exercise is to familiarize yourself with the basic syntax and structure of C++ programming. You will write a simple program that demonstrates the use of variables, data types, and basic input/output operations.

## Instructions
1. **Create a new C++ file** named `exercise1.cpp`.
2. **Write a program** that:
   - Declares variables of different data types (e.g., `int`, `float`, `char`, `string`).
   - Assigns values to these variables.
   - Outputs the values of the variables to the console.
   - Takes user input for one of the variables and displays it.

3. **Example Output**:
   ```
   Enter your name: John
   Hello, John! Welcome to C++ programming.
   ```

4. **Submit your code** by uploading the `exercise1.cpp` file to the designated submission area.

## Hints
- Use `#include <iostream>` for input and output operations.
- Use `std::cout` for output and `std::cin` for input.
- Remember to compile your program using a C++ compiler (e.g., g++) before running it.
