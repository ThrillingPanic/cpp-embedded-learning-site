# Exercise 2: C++ Basics

## Task

Write a C++ program that demonstrates the use of control structures. Your program should include the following:

1. **Input**: Prompt the user to enter a number.
2. **Control Structure**: Use an `if` statement to check if the number is even or odd.
3. **Output**: Display a message indicating whether the number is even or odd.

## Example Output

```
Enter a number: 4
The number 4 is even.
```

```
Enter a number: 7
The number 7 is odd.
```

## Requirements

- Use appropriate data types for variables.
- Ensure that your program handles invalid input gracefully (e.g., non-integer values).
- Comment your code to explain the logic behind your control structures.

## Submission

Submit your code as a `.cpp` file named `exercise2.cpp` in the `public/examples` directory.
