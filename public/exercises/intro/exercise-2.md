# Exercise 2: Introduction to C++ and Embedded Systems

## Objective
The goal of this exercise is to reinforce your understanding of the basic concepts of C++ and how they apply to embedded systems.

## Tasks

1. **Basic Syntax**
   - Write a simple C++ program that prints "Hello, Embedded World!" to the console. 
   - Ensure you include the necessary headers and use the `main` function correctly.

2. **Variables and Data Types**
   - Create a program that declares variables of different data types (int, float, char, and bool).
   - Assign values to these variables and print them to the console.

3. **Control Structures**
   - Write a program that uses an `if` statement to check if a number is even or odd.
   - Use a loop to print all even numbers from 0 to 20.

4. **Functions**
   - Define a function that takes two integers as parameters and returns their sum.
   - Call this function in your `main` function and print the result.

## Submission
- Save your code in a file named `exercise-2.cpp`.
- Ensure your code is well-commented to explain each section.
- Submit your code for review.
