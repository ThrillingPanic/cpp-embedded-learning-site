# Exercise 1: Introduction to C++ and Embedded Systems

## Objective
The goal of this exercise is to familiarize yourself with the basic concepts of C++ and embedded systems. You will learn about data types, control structures, and the fundamental principles that govern embedded programming.

## Instructions
1. **Data Types**: 
   - Research and list at least five different data types in C++. 
   - For each data type, provide a brief description and an example of how it is used in a program.

2. **Control Structures**:
   - Write a simple C++ program that demonstrates the use of at least one conditional statement (if-else) and one loop (for or while).
   - Explain how these control structures help in making decisions and repeating actions in your program.

3. **Embedded Systems Basics**:
   - Define what an embedded system is and provide two examples of embedded systems in everyday life.
   - Discuss the importance of understanding C++ in the context of embedded systems.
